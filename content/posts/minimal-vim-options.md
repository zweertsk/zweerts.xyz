Bare minimum list of options to add to a new vimrc file.
```
syntax on
filetype plugin indent on

set cursorline
set ruler
set relativenumber
set tabstop = 2
set shiftwidth = 2
set expandtab
set wrap
set noswapfile
set nobackup
set nohlsearch
set incsearch
set ignorecase
set smartcase
set scrolloff = 8
set colorcolumn = 80
```
